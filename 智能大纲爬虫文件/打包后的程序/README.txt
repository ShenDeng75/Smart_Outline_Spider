1，目录说明：
     depend：--JieBa分词的依赖文件。
	 SkillFile：
	     关键字.json：--用户选择后的技能点。
		 关键字-高频词汇.json：--高频词汇。
		 关键字-技能点.json：--原始技能点。
		 关键字-目标文件,json：--最终技能点。
	idf.txt：--JieBa分词的依赖文件。
	Job51.exe：--第一步程序。
	Merge.exe：--第二步程序。
	
2，使用步骤：
     在命令行中
	   1>> Job51 关键字
	   2>> Merge 关键字
	   
3，"关键字.json"文件约定：
     1)位置必须放在SkillFile文件夹下。
	 2)命名必须以"关键字.json"命名。
	 3)内容必须是["A|B|C", ]格式。
	 4)编码格式为'utf-8'。
